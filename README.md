# Тестовое задание на вакансию Junior Golang-разработчик

## Инструкция по запуску

**Требования**:
- Установлен Go (рекомендуется версия 1.25).
- Установлен Docker Desktop.
- Установлен инструмент `migrate` для применения миграций.
- Установлен `make` для использования Makefile.

1) Подготовка файла `.env`
- Скопируйте шаблон и заполните параметры (находится в `configs/env.example`).
- Файл нужно положить в `configs/.env`.

По умолчанию в `configs/env.example` указаны тестовые параметры (user: `user`, password: `1234`, db: `subscriptions_db`).

2) Сборка и запуск, остановка и удаление контейнеров через docker compose
- В корне проекта в терминале выполните для сборки и запуска контейнеров:
```bash
make build
```
- Для остановки и удаления контейнеров:
```bash
make stop
```

3) Миграции базы данных
- В проекте имеются SQL-миграции в `internal/migrations`. Их можно применить с помощью утилиты `migrate`:
```bash
make up
```
- Аналогично для отката:
```bash
make down
```

4) Swagger
- Для генерации swagger-спецификации выполните в терминале:
```bash
make swagger
```
